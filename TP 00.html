1. ¿A qué nos referimos con los elementos títulos?

Los elementos títulos o encabezados son cadenas de texto que decriben brevemente el tema de la sección que representan. Existen 6 tipos de elementos de encabezado, siendo h1 el más importante, y h6 el menos importante.

2. ¿Cuál es la importancia del número que lleva la etiqueta de título?

El número que lleva la etiqueta de título es importante porque determina la importancia, y pot lo tanto, el tamaño del título. Así, h1 será el título mas grande e importante, mientras que h6 será el título más pequeño y menos importante.

3. ¿Qué es lo que debe evitarse para que el uso de los títulos sea accesible a las personas no videntes?. Justificar

Lo que debe evitarse para que el uso de los títulos sea accesible a las personas no videntes es saltearse níveles de etiquetas y confundir a los usuarios. Por el contrario, una buena estructura de níveles de encabezado permite que las personas no videntes se muevan y conozcan el contenido en la pantalla facilmente y en poco tiempo.

4. ¿Cuál es el concepto de anidar títulos?. Ejemplificar con un ejemplo distinto al propuesto por mdn.

El concepto de anidar títulos consiste en añadir etiquetas dentro de otras, generando subsecciones en nuestros documentos. Esto permite una mejor organización de la información y ayuda a los usuarios a conocer las jerarquías de contenidos. 5. ¿Qué es HTML y en qué consiste?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para desarrollar una página web y estructurar sus contenidos, que pueden ser párrafos, listas, imágenes o tablas de datos. HTML consiste en una serie de elementos que usas para encerrar distintas partes de contenido y que se vean o comporten de una determinada manera.

6. ¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minuscula?

Una etiqueta es un fragmento de código que permite crear y dividir elementos HTML. Funcionan de igual forma, tanto si se escriben en mayúscula o en minuscula.

7. ¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

Un elemento HTML es una porción de contenido dentro del código.

Las partes principales del elemento HTML son:

Etiqueta de apertura

Es el nombre del elemento, encerrado por parentesis angulares.

Etiqueta de cierre

Igual que la de apertura, pero incluye una barra de cierre antes del nombre de la etiqueta.

Contenido

Lo que contiene el elemento, generalmente es texto.

Elemento

La etiqueta de apertura, de cierre y el contenido conforman el elemento en sí.

8. ¿Qué es un atributo? Dar 2 ejemplos.

Los elementos también pueden contener atributos, que contienen información adicional sobre el elemento, que no aparecerá en el contenido real del elemento. Cada atributo tiene un nombre y un valor, asignado con el signo "=".

Algunos ejemplos de atributos son:

class:

Permite darle al elemento un nombre identificador, que sirve para apuntarle información de estilo y otras cosas.

content:

Permite que el sitio web se adapte a distintos tamaños de dispositivos.

12. Especificar el uso de las siguientes etiquetas: blackquote, cite, hr, abbr, code, span, a.
Ejemplificar.

La etiqueta <p></p>

blockquote

Crea citas en bloque de otros autores o documentos.
Ejemplo:
<blockquote cite='http://html.conclase.net/w3c/html401...def-BLOCKQUOTE'></blockquote>

cite

Marca una referencia a una fuente, o el autor de un texto citado.
Ejemplo:
<cite>Galileo</cite> dijo: "... y sin embargo, se mueve."

hr

Representa un cambio de tema entre párrafos.
Ejemplo:
<p>Este es un parrafo unicamente visual para ver el uso de
la etiqueta hr
</p>

<p>Este es el segundo parrafo, separado del primero por la etiqueta hr, que puede
representarse mediante una línea horizontal.
</p>

abbr

Representa una abreviación o acrónimo.
Ejemplo:
<p>I do <abbr title="Hypertext Markup Language">HTML</abbr></p>

code

Marca el código en un programa.
Ejemplo:
<code>
#include main() { for(;;) { printf ("Hello World!\n"); } }
</code>

span

Sirve para aplicar estilo al texto o agrupar elementos en línea.
Ejemplo:
<span style="color: red;"></span>

a

Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página.
Ejemplo:
<a href="https://www.mozilla.com/">Enlace externo</a>
